# Geometric Localization of Waves on Thin Elastic Structures

## Usage

The programs and scripts in this repository require the standard Python
scientific stack ([NumPy][numpy] + [SciPy][scipy]) and
[Dedalus][dedalus].  Plotting requires [Matplotlib][mpl] and
[charu][charu].
